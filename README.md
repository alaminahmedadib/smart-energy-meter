# ⚡ Smart IoT-Based Energy Monitoring System with AI Prediction

## 📌 Overview
This project is a smart IoT-based energy monitoring and control system using ESP32.  
It measures real-time electrical parameters such as voltage, current, power, and energy consumption (kWh), and sends data to the cloud using MQTT.  
A web dashboard is used to visualize data and provide AI-based predictions.

---

## 🚀 Features
- 🔌 Real-time energy monitoring (Voltage, Current, Power, kWh)
- 📡 MQTT-based communication
- ☁️ Firebase cloud data storage
- 📊 Web dashboard with charts (daily & weekly)
- 🔮 AI-based next usage prediction
- ⚠️ Peak consumption detection
- 🌱 Energy usage insights (Low / Medium / High)
- 💡 Load control (3 devices via ESP32 + mobile app)
